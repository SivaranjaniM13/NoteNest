\# NoteNest - Firebase MBaaS App



A full-stack mobile + web app built with Firebase.



\## Features

\- Email/Password Authentication

\- Google OAuth Login

\- Firestore CRUD Operations

\- Firebase Storage (Image Upload)

\- Push Notifications (FCM)

\- Real-time Updates

\- Firebase Hosting



\## Setup



\### Android

1\. Clone this repo

2\. Add your `google-services.json` to `app/` folder

3\. Run in Android Studio



\### Web

1\. Add your Firebase config to `web/index.html`

2\. Run `firebase deploy --only hosting`
